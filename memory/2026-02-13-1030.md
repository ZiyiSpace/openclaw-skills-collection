## 💡 通用的 UI 设计技巧

### 1. 配色方案
- 使用**主色 + 辅助色 + 强调色**（3色原则）
- 小红书：紫 + 白 + 金色
- 避免超过 3-4 种主要颜色

### 2. 字体层级
- **标题**：大号、加粗
- **副标题**：中号、常规
- **正文**：小号、常规
- 保持字体大小比例：1.5 : 1.25 : 1

### 3. 间距系统
- **元素间距**：8px、16px、24px（使用 8 的倍数）
- **内边距**：16px、24px
- **外边距**：32px、48px

### 4. 视觉层级
- **层级 1**：大标题、强对比
- **层级 2**：正文、标准对比
- **层级 3**：辅助信息、弱对比

### 5. 对齐和网格
- 使用 4px 或 8px 网格对齐
- 同类元素左对齐或居中对齐
